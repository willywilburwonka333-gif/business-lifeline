# Business Lifeline

Business Lifeline is an AI-assisted Business MRI and rescue-planning tool for struggling small businesses.

## MVP features

- Multi-step business assessment
- Deterministic financial health calculations
- Business health score and runway estimate
- Prioritised rescue actions for today, 7 days, 30 days, and 90 days
- Demo business workflow
- Local persistence, print report, and restart controls
- Safety warnings and professional-advice disclaimer

## Local development

```bash
npm install
npm run dev
```

Open the forwarded port 3000 URL when using GitHub Codespaces.

## Validation

```bash
npm test
npm run lint
npm run build
```
