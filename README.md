# Business Lifeline

Business Lifeline is an AI-assisted Business MRI, recovery planner, and lightweight operating system for struggling small businesses.

It turns raw business figures into an immediate diagnosis, a practical rescue pathway, and a persistent workspace the owner can use to execute the turnaround.

## What it does

- Guided Business MRI assessment
- Deterministic financial calculations and safety checks
- Health score, operating margin, cash runway, debt pressure, and receivables pressure
- Prioritised actions for today, 7 days, 30 days, and 90 days
- AI-enhanced analysis grounded in the supplied business data
- Recovery timeline and specialised recovery playbooks
- Recovery Coach for follow-through and recovered-value tracking
- Business Brain for contextual questions about the saved MRI
- Multi-lever cashflow simulator
- Business Operating System for tasks, contacts, responsibilities, targets, and document controls
- Printable action sheet and practical communication templates
- Local persistence, mobile-responsive tabbed workspace, print support, and restart controls

## Safety approach

Financial calculations and urgent-risk detection are deterministic. AI is used to improve explanation and prioritisation, not to replace accounting, legal, financial, or insolvency advice. The interface escalates serious tax, payroll, legal, debt, and closure indicators to qualified professional support.

## Local development

```bash
npm install
npm run dev
```

Open the forwarded port 3000 URL when using GitHub Codespaces.

## Final validation

```bash
npm run check
```

This runs the complete automated test suite, ESLint, and a production Next.js build. The same quality check runs through GitHub Actions on every push to `main`.
